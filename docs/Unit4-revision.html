Structure of Internet

WAN vs LAN

  • WAN is over larger geographical area
  • WANs use 3rd party equipment.

Components of a URL

  • Protocol – Examples are http:// or https://
  • Domain name – Examples are theimportanceofhomework.co.uk
    • Top Level Domain (TLD) is “.uk”
  • /homework/sheet5.html is the resource required from the server

Domain Name Service

  • Associates domain names with IP addresses
  • DNS Servers are hierarchical, root DNS servers responsible for serving the definitive values.
  • Internet Registry – Organisation that controls allocation of domains to IP addresses.

WWW vs Internet

  • Internet describes connections between computers
  • Internet covers specific protocols (TCP/IP)
  • WWW refers to the set of hyperlinked documents, accessed using the HTTP/HTTPS protocols over the Internet

Wireless Networks

  • NIC – Network Interface Card
  • WAP – Wireless Access Point

Internet Communication

  • Circuit Switching – Creating a channel between two endpoints that endures during transaction (phone call)
  • Packet Switching – Data is broken into small pieces and sent via many different routes, reassembled upon reception.
  • Protocols – A set of rules for data communication, they must be standardised across devices for communications to work.

Layers

Application Layer

Defines the set of standards used to communicate across the internet. Examples:
  • HTTP/HTTPS – Web Browsing, S = secure = encryption
  • Email
    • SMTP – Simple Mail Transfer Protocol – sending emails
    • POP3 – Downloading emails, deleted from server
    • IMAP – Reading emails, persisted on the server
  • FTP – File transfer Protocol
  • SSH – Secure Shell

Transport Layer

Splits data into packets, numbers them sequentially. Adds a port number to identity the destination service. TCP/IP – Transmission Control Protocol

Internet Layer

Give packets the address of the source and destination
  • IPv4 – 32-bit number, 4 decimal numbers from 0-255 separated by ‘.’
  • IPv6 – 128-bit number, 8 hexadecimal numbers from 0000 - FFFF, separated by ‘:’
Routing tables used to instruct the next hop as packet travels across the internet.

Link Layer

Operates the physical connection between any given set of two devices.

Every device has a MAC address – 6 hexadecimal numbers from 00 – FF, separated by ‘:’ Devices

  • Switch – Can route packets within a private IP network, based on the MAC address
  • Router – Can route packets between distinct IP networks (for example between a private LAN and the public internet)
  • Gateway – Can route packets between networks that use distinct protocol stacks (for example between the mobile phone network and the internet).

Network Threats

Threats

  • Malware – Malicious Software
    • Viruses – Attach to host files/programs and propagate when executed opened
    • Worm – Standalone program that can run and propagate itself
    • Trojan Horse – Malware hidden inside an apparently desirable payload, such as a free game/free software (pirated software is a haven for these)
  • Phishing – Sending emails/text messages with links to malicious sites to either
    • Request further private details from duped users
    • Run malicious code in the rouge website once visited

Protection & Detection

  • Code Quality
    • Guarding against buffer overflow attack
    • Guarding against SQL injection
  • System Access
    • Strong passwords, password policy
    • Two factor Auth
    • User Access Rights
  • Encryption
  • Firewall
    • Software that filters packets based on port numbers/IP addresses
    • Packet filtering based on content
  • Proxy Server
    • A server within a large enterprise that makes requests on users behalf
    • Allows for site filtering (school uses one of these)
    • Can provide anonymous serving (to some degree)
    • Logs user access
    • Provides caching, out of date cache = stale
  • Monitoring
    • Packet sniffers
    • User Access logs, who, what, where, when & why?
  • Anti-virus & Anti-malware software
    • Using up-to date signatures of known malware to scan all incoming files/visited sites
